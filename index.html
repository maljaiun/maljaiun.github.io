Set up VPS (bhw) part 1

I puchased one year bhw vps, I’ll set up this VPS, do some simple default and record it.

Ubuntu I chosse Ubuntu as my VPS’s OS.

  1. open bwh1.net then click Services>my services>KiwiVM Control Panel>install new OS
  2. then choose Ubuntu 16.04 and I received email contains a root user and passwd.

  3. ssh login root user then reset root passwd by type command

  4. add user as sudo

  5. Sudo not using passwd

  6. set up ssh-key login

  7. login by simmple command

then reboot ssh service sudo systemctl restart sshd.service
